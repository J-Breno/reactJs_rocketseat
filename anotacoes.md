
### ESLint

#### instalação

```
npm i eslint -D
npm i @rocketseat/eslint-config -D
```

crie um arquivo chamado: .eslintrc.json
para testar e já corrigir  use:

```
npx eslint src --ext .ts, .tsx --fix
```
